# domzack-chart-demo

Demo mínima da `domzack-chart-lib` contendo apenas o essencial para o usuário final:

- `public/index.html`
- `public/css/styles.css`
- `public/js/main.js`
- `README.md`

## Como usar

1. Abra `public/index.html` no navegador.
2. Pronto: a biblioteca é carregada via CDN.

## Estrutura

```text
public/
  index.html
  css/styles.css
  js/main.js
README.md
```

## CDN usado no HTML

```html
<script src="https://cdn.jsdelivr.net/gh/domzack/domzack-chart-lib@main/dist/index.global.js"></script>
```

## Observação

Se quiser rodar com servidor local (opcional), use qualquer servidor estático de sua preferência.
